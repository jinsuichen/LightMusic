# ✨ 特色

- 🚢 跨平台构建，支持 Windows、Linux、macOS
- 📐 界面简洁，以少胜多
- 🚀 主要使用 <a href="https://reactjs.org/">React</a>、 <a href="https://vitejs.dev/">Vite</a> 和 <a href="https://www.electronjs.org/">Electron</a> 等技术

# ⚙ 运行代码

## Clone 这个仓库

```bash
$ git clone https://github.com/jinsuichen/LightMusic.git
```

## 安装依赖包

```bash
$ npm install
```

## 开发模式运行

```bash
$ npm run dev
```

## 打包

```bash
# For Windows
$ npm run build:win

# For macOS
$ npm run build:mac

# For Linux
$ npm run build:linux
```
